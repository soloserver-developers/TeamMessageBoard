# TeamMessageBoard

**このアドオンはSoloServerCore3に統合されました。**

TeamMessageBoardはSoloServerCore3のアドオンとして動作するメッセージボードプラグインです。  
このプラグインはチームで共有できるメッセージボードを提供します。

## 動作要項

- Spigot 1.15.x
- MySQL or MariaDB 5.x ~

### 前提プラグイン

- SoloServerCore 4.x ~
